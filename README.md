# text-to-html
Convert Markdown notes from Obsidian, Joplin, or ChatGPT into a clean, responsive HTML website. Supports headings, bold, lists, code blocks, tables, and emojis with live preview and one-click HTML copy. Perfect for creating documentation, DevOps notes, or quick static web pages.

# 🧠 Markdown → HTML Website Generator

I built this tool to convert my personal notes into a clean HTML webpage. It is fully open source, so anyone can use it for their own projects or websites. Contributions, ideas, and feature improvements are always welcome.

I usually write notes in:
- Obsidian
- Joplin

These notes use Markdown (headings, bold, tables, etc).  
Instead of manually writing HTML, I just paste the content into this tool and generate a full website instantly.

---

## 🚀 Why I built this

When copying notes directly into HTML:

- Formatting gets messy ❌  
- Tables don’t render properly ❌  
- Emojis break ❌  

So I created this simple generator to fix all of that.

---

## ✨ Features

- Convert Markdown → HTML
- Live preview
- Copy full HTML code
- Responsive layout (mobile + desktop)

### Supported formatting

- Headings (#, ##, ###)
- Bold (**text**)
- Italic (*text*)
- Inline code (`code`)
- Code blocks (```)
- Lists
- Tables
- Horizontal lines (---)
- 💡 Notes (special highlight box)

---

## 🛠️ How I use it

1. I write notes in Obsidian like this:
2. Copy the content  
3. Paste into this tool  
4. Click **Generate**  
5. Preview is shown  
6. Copy HTML  
7. Save as `.html`

Done ✅
